How to run the server:

1.Turn on your mysql first.
2.Import the project into your java IDE.
3.Run CreateDB.java to create the server database.
4.Run MyServer.java. At this time the server is working, and the IPAddress and port number is shown in the console window in your IDE.
5.Run the app on your android phone, input the IPAddress and port number of Server, then you can freely play with it.
